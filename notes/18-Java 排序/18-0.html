1. 插入排序 Insertion Sort 2. 冒泡排序 Cocktail Sort 3. 合併排序 Merge Sort 稳定、快速的排序算法 4. 快速排序 Quick Sort 5. 堆積排序 Heap Sort -> 完全二元樹 complete binary tree 6. 桶排序 Bucket Sort 7. 基数排序 Radix Sort 快速排序、 插入排序、 选择排序、 冒泡排序、 的 最差时间复杂度 为 O(n2)。 归并排序的 平均情况 和 最差情况 的复杂度 是O(nlogn)。 快速排序的 平均时间 是O(nlogn)。 堆排序的 时间复杂度 是O(nlogn)。 桶排序 和 基数排序 都是针对键值的特定排序算法。 这些算法不是通过比较键值而是使用桶来对键值排序的，他们会比一般的排序算法效率更高。
