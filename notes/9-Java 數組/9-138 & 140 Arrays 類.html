說明： -> Arrays 是 JDK 的一種類，裡面包含了很多種方法 1. -> Arrays.sort(int[] list, fromIndex, toIndex) -> 對部分參數進行排序（1-2 不包括3） -> ex: Arrays.sort(list); // 對全部內容進行排序 2. -> Arrays.binarySearch(int[] a, int key) -> 傳入array，並查找相對應的參數,並返回位置key值(int) -> 無找到，返回 -1 3. -> Arrays.equals( int[] a, int[] b) -> 比對內容 -> a == b 是比對記憶體位置 4. -> Arrays.toString( int[] a ) -> 將 array 遍歷並用 string 顯示 -> Arrays.toString(int[] list) demo： -> src/main/java/arraydemo/ArraysClassDemo.java 1. 2.
