demo : src/main/java/sort/BubbleSort.java 2. 氣泡排序法 [2, 9, 1, 4, 8, 3] => [1, 2, 3, 4, 8, 9] 較小的值會 浮到 頂部，較大的值會 沈入 底部 當整個迴圈跑完，最後一個數將會是 整個數組中最大的數 說明： 氣泡排序法的概念很容易，分為幾個步驟 1. 比較相鄰的元素。如果第一個比第二個大，就交換。 2. 對每一對相鄰元素作同樣的工作，從開始第一對到結尾的最後一對(這時候最後一個數應該會是最大) 3. 氣泡排序工作原理：兩個值相互比較，一直到 第二個值 大於 第一個值 break 跳出迴圈 時間複雜度: O(n^2) 圖解： 1, 4, 2, 9 ---> [ 1, 4, 2, 9 ] 1, 4, 9, 2 1, 2, 4, 9 1, 9, 4, 2 1, 2, 4, 9 -> [9, 1, 4, 2] 面試： 氣泡排序法 -> 是否為 最穩定的算法 ? -> 複雜度太高效率非常不好(Ο(n^2)) -> 存在 最佳情況 和 最差情況 -> 可能數組本身就已經排好序了，不要sort 如果本身是沒有排序的，那就需要進行多個比較
