demo： src/main/java/sort/QuickSort.java 4. 快速排序 工作原理： -> 目前公認最快的排序法，雖然最壞的時候還是會衝到O(n^2) 1.先將最左邊的數 X 當成基準點(s),也稱為 主元(pivot) 2.索引 I 由左邊向 右 找，一直找到大於基準點(s)的數 3.索引 J 由右邊向 左 找，一直找到小於基準點(s)的數 最差複雜度: O(n^2) 平均複雜度: O(nlog^n) => O(n) 圖解： -> [9, 1, 4, 2, 7, 11] -------------------------------------------------------------------- -> 9 (主元) [1, 4, 2, 7] [11] // 比9小在左邊, 反知 -------------------------------------------------------------------- -> 1 (主元) // 9 [] [4, 2, 7, 11] -------------------------------------------------------------------- -> 4 (主元) // 1, 9 [2] [7, 11] -------------------------------------------------------------------- -> 2 (主元) // 1, 4, 9 [] [7, 11] -------------------------------------------------------------------- -> 7 (主元) // 1, 2, 4, 9 [] [11] -------------------------------------------------------------------- -> 11 (主元) // 1, 2, 4, 7, 9 [] [] -------------------------------------------------------------------- -> 1, 2, 4, 7, 9, 11
