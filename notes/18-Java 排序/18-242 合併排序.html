demo : src/main/java/sort/MergeSort.java https://magiclen.org/merge-sort/ 3. 合併排序 說明： 先將一筆array資料拆除2筆, 並且進行排序，一直拆解到只剩下 "[x]" 最後才進行合併。 兩筆已排序的資料合併並進行排序， 如果所讀入的資料尚未排序，可以先利用其它的排序方式來處理這兩筆資料，然後再將排序好的這兩筆資料合併。 圖解： [3,1,4,6] -> 原始資料 [3,1] [4,6] -> 拆解 [3] [1] [4] [6] -> 在拆解 [1,3] [4, 6] -> 合併並排序 [1,3,4,6] -> 合併並排序
